PyWince Executable¶

Plays .wav files and lists connected sound input/output devices as well as connected FTDI USB devices.

main.int_handler(*args)¶

SIGINT and SIGTERM interrupt handler that cancels all asyncio tasks.

main.main(filename, device=1)¶

Transforms input .wav file to a numpy.ndarray object for sound buffer. Calls main.play_file() with the sound buffer.

Parameters

  • filename (str) – The file name of the sound file.

  • device (int, optional) – The device number of the output sound source, defaults to 1.

main.play_file(buffer, device)¶

Asynchronously plays input sound buffer.

Parameters

  • buffer (numpy.ndarray) – Input sound buffer used by sounddevice.OutpuSttream.

  • device (int) – Output sound device number.

Raises

sd.CallbackStop – Exception to be raised by the user to abort callback processing. All pending buffers are discarded and the callback will not be invoked anymore. See sounddevice.CallbackStop.
